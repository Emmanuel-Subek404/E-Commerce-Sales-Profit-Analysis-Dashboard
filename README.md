# E-Commerce Sales & Profit Analysis Dashboard
Project Overview

This project is an interactive business intelligence dashboard built using Microsoft Power BI and MySQL to analyze sales, profit, customer behavior, product performance, and business risks in an e-commerce business.

The dashboard focuses on:

Revenue analysis
Profitability analysis
Customer insights
Product performance
Risk & growth opportunities
Geographic profitability analysis

The goal of this project is to transform raw CSV sales data into meaningful business insights that support strategic decision-making.

This dashboard answers important business questions such as:

Key Insights

Some major business insights discovered:

Electronics generated the highest revenue overall.
Clothing maintained the highest profit margin.
Several products generated high revenue but low profitability.
Certain cities consistently outperformed others in profit generation.
Loss-making products significantly impacted overall margins.
Top customers contributed a major share of total business profit.
